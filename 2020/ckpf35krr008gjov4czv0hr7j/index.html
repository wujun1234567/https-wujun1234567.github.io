1.对应第三方模块的版本不合适的话，安装的时候需要指定版本安装。
版本的过高或过低有可能会出现运行出错，所以有时候需要安装的指定版本、

解决方案：
例如 npm install mine@2.1.2 指定安装 mine模块的 2.1.2的版本
