想要让git对一个目录进行版本控制需要以下步骤

  • 进入要管理的文件夹
  • 执行初始化命令
  • 管理目录下的文件状态
  • 管理指定文件（红变绿）
  • 个人信息配置：用户名、邮箱【只需配置一次即可】
  • 生成版本
  • 查看版本记录
以下总结：

  1. 进入要管理的目录

  2. git init 初始化，即：让git帮助我们管理当前文件夹

  3. git status 检测当前目录下文件的状态

  4. 三种状态的变化

  • 红色：新增的文件/修改了原老文件 –> git add 文件名
  • 绿色：git 已经管理起来 —> git commit -m ‘描述信息’
  • 生成版本
